# maven-learn
Maven学习代码笔记

## Issues
使用的时候遇到任何问题或有好的建议，请点击进入[issue](https://github.com/agile-contrib/maven-learn/issues)，欢迎参与维护！

## How to use?

首先，需要进行打包：

```
mvn clean package
```

然后，命令行进入```target/classes```后执行：

```
java io.gitee.hai2007.mavenlearn.App
```
